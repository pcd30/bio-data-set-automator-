# Bioinformatics Dataset Automation Pipeline

## Overview

This project implements an automated pipeline for generating and analyzing DNA sequence datasets using Python. The workflow demonstrates core bioinformatics tasks such as sequence generation, FASTA file parsing, GC content analysis, motif detection, and dataset visualization.

The project is implemented in **Google Colab** and uses **Biopython** for biological sequence processing.

---

## Features

* Generate synthetic DNA sequences
* Store sequences in FASTA format
* Parse biological sequences using Biopython
* Compute GC content for each sequence
* Visualize GC content distribution
* Detect DNA motifs (ATG start codon)
* Calculate dataset statistics
* Export structured analysis results to CSV

---

## Technologies Used

* Python
* Biopython
* Pandas
* Matplotlib
* Google Colab

---

## Project Workflow

1. Generate random DNA sequences
2. Save sequences to a FASTA dataset
3. Parse sequences using Biopython
4. Calculate GC content for each sequence
5. Visualize GC content distribution using a histogram
6. Detect DNA motifs within sequences
7. Compute datas
